Discuss the scope of var, let and const?

Key points are:

  • var declarations are function scoped while let & const are block scoped.
  • var variables can be updated and re-declared within its scope
    let variables can be updated but not re-declared
    const variables can neither be updated and re-declared
  • They all are hoisted in their own scope. var is initialized with undefined but let and const variables are not initialized.
Tell us the use case of null and undefined?

Key points are:

  • null and undefined are primitive type in JavaScript
  • When we assign null as a value to any variable, it means that it is empty or blank
  • undefined means "not defined". So we declare a variable but do not assign a value to it, the variable becomes undefined.
What do you mean by REST API?

A REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions for building and interacting with web services or web applications.

Imagine a REST API as a menu in a restaurant. The menu lists various dishes (resources), and each dish has a name (URL) and a description (data). When you (the client) want to order something, you look at the menu, choose a dish, and tell the waiter (the server) what you want. The waiter then brings you the dish you ordered.

  • The menu is like the API, listing available resources and their URLs.
  • Choosing a dish is like making an HTTP request (e.g., GET, POST, PUT, DELETE) to a specific URL.
  • The dish you receive is the response from the API, which can include data in various formats like JSON or XML
